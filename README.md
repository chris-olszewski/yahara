## The Yahara Project

Making local music super awesome since 2014 :)


### Setup
1. Make sure node is installed (`brew install node` on OSX)
2. Install grunt command line `npm install -g gulp`
3. git clone
4. cd into directory
5. `npm install`
6. `bower install`
7. `gulp`
8. visit http://localhost:8000 in a browser

Built using:
- Ember
- Node.js
- Gulp
- Express
- Bower
